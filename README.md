## Build
```
$ cd se-05-team-09/
$ mkdir build
$ cd build/
$ cmake .. && make
$ ./pencilproducer
```
## Documentation
Prebuilt documentation can be read by:
```
$ cd doc/html
$ browser index.html
```

## Testing
Test cases can be found in:
```
$ Test/tst_test.cpp
```
### Build Test Cases
```
$ cd Test/
$ mkdir build/
$ cd build/
$ cmake .. && make
$ ./testpencils
```
